# Usage

### Build it with:
`bun --bun run build`

### Serve it with
`bun --bun run start`

# DB Design
4 collections in total
1. Boards - [Model file](./lib/models/board.ts)
2. Column - [Model File](./lib/models/column.ts)
3. Job Application - [Model file](./lib/models/job-applications.ts)
4. User Collection - Generated by Better-Auth automatically

# Notes
1. Only used [middleware](./proxy.ts) to restrict user from accessing sign up and sign in pages after having logged in. If issues encountered regarding login and logout, check out the proxy file.
2. Uses MongoDB Atlas for managed DB but connection errors occur occasionally. So, check your DNS setting and VPN connection before logging in or signing up.
